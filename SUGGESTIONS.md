1. **Wire production-ready auth path**
   - Finalize login/register and token validation flow across backend and frontend.
2. **Complete Stripe checkout flow**

3. **Finish admin control surface**
   - Add role-aware admin management for routing policies and user/account operations.
4. **Implement source/destination templates**
   - Provide ready-to-run templates for common routing stacks.
5. **Improve deployment automation traceability**
   - Keep Vercel test URL, deploy policy, and validation artifacts linked in docs.

## Suggestion scoring model

- **Impact:** user or revenue lift potential
- **Effort:** implementation complexity and integration cost
- **Risk:** operational and security risk if delayed

Use the highest-impact, lowest-risk sequence first in each implementation pass.
